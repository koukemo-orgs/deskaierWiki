# DeskAIer Wiki

Documentation repository for DeskAIer GitHub Pages.

## Structure

- `ja/` Japanese docs
- `en/` English docs
- `index.md` language selector

## Publish with GitHub Pages

1. Open **Settings > Pages** in this repository.
2. Source: **Deploy from a branch**
3. Branch: `main` / folder: `/ (root)`
4. Save.

GitHub Pages will build from Jekyll files in this repo.
