# Home UI & Display Settings

## Section visibility

You can toggle each section on/off individually:

- Home widgets
- AI quick launch
- Smart suggestions
- Web shortcuts
- Pinned section
- Hidden folder

## View mode (List / Icon)

Per-section view mode is available for:

- Web shortcuts
- Smart suggestions
- Pinned section

If a section is OFF, its dependent view controls are hidden automatically.

## Group app view mode

- List
- Icon
- Ring
